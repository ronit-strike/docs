## Welcome to Strike

At **Strike**, we’re revolutionizing the world of sports betting by treating sports bets as their own asset class. Our innovative platform empowers users to place, manage, and analyze sports bets in a dynamic, intuitive environment. Whether you're building a new application or integrating sports betting features into your existing platform, our API provides you with the robust tools you need.

This documentation is powered by our OpenAPI specification, offering a complete guide to all available endpoints, including:

- **Betting Events:** Retrieve upcoming or past betting events with flexible filtering options.
- **Players:** Access detailed information about athletes, including stats and profiles.
- **Lineups & Bets:** Create and manage lineups, placing sports bets with validation and risk management.
- **Payments & Withdrawals:** Handle financial transactions securely and efficiently.
- **Player Statistics:** Retrieve comprehensive statistics to help inform your betting strategies.


For this documentation, we are using our OpenAPI specification to ensure a consistent and comprehensive reference for every endpoint.

## Authentication

All API endpoints are secured and require authentication via API keys. Every request must include your unique API key in the `X-API-Key` header. This authentication mechanism protects our services and ensures that only authorized users can access Strike’s innovative sports betting features.
